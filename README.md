# Website

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## Edit
```
git clone git@github.com:Web3-Jerry/mainnet.git 
git checkout -b main origin/main
...
git add .
git commit -m"update doc"
git push
```

## Preview locally

```bash
npm run start
```

## Publish
```
git checkout main
npm run deploy
```
